SOS Board Game

A combinatorial game

Quick Tech Facts

  • Developed entirely programmatically in Swift and UIKit
  • Custom MiniMax implementation
  • Firebase for multiplayer support
  • Modular architecture
  • Download for iOS
  • Learn More

About

SOS is a combinatorial game when played with two ore more players. It is similar to tic-tac-toe, dots, and boxes, but has greater complexity. In terms of game theory, it is a zero-sum, sequential game with perfect information.
