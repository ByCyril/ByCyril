My mini-workshops this week!

April 15, 2021 - April 16, 2021

I have the awesome privilage of be giving two technical talks on topics I have great interest and passion for: The Swift Programming Language and Machine Learning for iOS.

You can checkout the Eventbrite page below to register as well as checking out other awesome speakers.

Eventbrite


Hello, Swift

April 15, 2021 - 10:00am - 10:45am PST

In this workshop, I hope to give Swift the spotlight it deserves and talk about what makes Swift a unique language, by demonstrating some of the cool features it provides from typecasting to functional programming. By the end of my presentation, the audience will take with them a new appreciation for the language and the opportunity to write elegant Swift code for their future projects to come.

Machine Learning for iOS

April 16, 2021 - 12:00pm - 1:00pm PST

Core ML works with other high level libraries like Vision for image and object detection, natural language for language processing, and GameplayKit for decision trees. Both libraries are easy to use and the audience will take away knowing how to create their own pre-trained classifiers to create an image recognition app and a sentiment analysis app.
